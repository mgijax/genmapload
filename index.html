Genetic Map Load (genmapload) Data and Logs

Data

  • Input Files
    • Genetic Map - from cgd.jax.org/mousemapconverter

  • Output Files generated from the Genetic Map Scripts
    • mgi_map.txt - all Marker cM positions/offsets
    • MRK_Offset.txt - new Marker cM positions/offsets

  • Output Logs
    • genmapload.cur.log - Curator's file
    • genmapload.proc.log - Proc file
    • genmapload.diag.log - Diagnostic file

  • Reports
    • DiffEqual.rpt - old map offset equal to new map offset
    • DiffFromNonSynToNonSyn.rpt - from non-syntenic to non-syntenic
    • DiffFromNonSynToSyn.rpt - from non-syntenic to syntenic
    • DiffFromSynToNonSyn.rpt - from syntenic to non-syntenic
    • DiffNotEqual.rpt - old map offset not equal to new map offset

  • Archive - Data files from previous loads.
