Genetic Map Load (genmapload) Data and Logs

Data

  • Input Files
    • Genetic Map (snpmap.csv) - from cgd.jax.org/mousemapconverter
    • MIT Map (MIT-marker-data.txt) - from cgd.jax.org/mousemapconverter

  • Output Files generated from the Genetic Map Scripts
    • mgi_map.txt - all Marker cM positions/offsets
    • MRK_Offset.txt - new Marker cM positions/offsets

  • Output Logs
    • genmapload.cur.log - Curator's file
    • genmapload.proc.log - Proc file
    • genmapload.diag.log - Diagnostic file

  • Archive - Data files from previous loads.
